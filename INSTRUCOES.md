# QuickHub - Instruções Rápidas

## Como Iniciar

### Opção 1: Usando o script
1. Execute `instalar.bat` (primeira vez)
2. Execute `iniciar.bat` (sempre que quiser usar)

### Opção 2: Manualmente
1. Abra o terminal na pasta do projeto
2. Execute: `npm install` (primeira vez)
3. Execute: `npm start`

---

## Como Usar

### 1. Abrir o Hub
- **Duplo clique** no ícone do tray (bandeja do sistema)

### 2. Adicionar Acessos
1. Clique no ícone ⚙ (Configurações)
2. Clique em "+ Novo Acesso"
3. Preencha:
   - Nome: Nome do app/site
   - Tipo: URL, Aplicativo, Pasta ou Arquivo
   - Caminho: Link ou caminho
4. Clique em "Salvar"

### 3. Mini Hub (Acesso Rápido)
- Pressione `Ctrl+Espaço` a qualquer momento
- Clique no acesso desejado

---

## Tipos de Acesso

| Tipo | Exemplo |
|------|---------|
| URL | `https://www.google.com` |
| Aplicativo | `C:\Program Files\Chrome\chrome.exe` |
| Pasta | `C:\Users\Documentos` |
| Arquivo | `C:\Relatorios\relatorio.pdf` |

---

## Atalhos

| Ação | Atalho |
|------|--------|
| Abrir Mini Hub | `Ctrl+Espaço` |
| Fechar Mini Hub | `ESC` |

---

## Dicas

1. **Organize por categorias**: Use o campo "Categoria"
2. **Use ícones**: Facilita a identificação
3. **Backup**: Exporte o `shortcuts.json`
